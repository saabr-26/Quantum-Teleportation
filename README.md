# Quantum Teleportation Simulation

This project simulates quantum teleportation using Qiskit in Python.

## How to Run

1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python main.py`
3. View results in the `results/` folder.

## Project Structure

- notebooks/: Interactive Jupyter experiments
- scripts/: Core simulation code
- results/: Plots and data
- slides/: Presentation
- report/: Final project report
